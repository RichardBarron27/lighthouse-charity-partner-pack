# Lighthouse — Charity Partner Pack

Everything a charity, school, or care home needs to distribute the **Still Here** crisis helpline browser extension.

Free. No signup required.

## What's Inside

- Print-ready posters and flyers
- Digital assets for websites and social media
- Installation guides for staff and volunteers
- Safeguarding integration documentation

## About Still Here

Still Here is a free browser extension that shows crisis helpline numbers when AI chatbot conversations turn dark. 100% local processing. Zero data collection.

## Get Started

Download this pack and distribute to your organisation. No technical setup required.
